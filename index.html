From First Lines of Code to Interactive Projects

I'm Hampton Miles, a high school junior who discovered coding as a way to bring creative ideas to life. What started with simple HTML tags has evolved into building interactive games, AI experiments, and polished web experiences. For me, coding isn't just about writing lines—it's about solving puzzles, creating beauty in logic, and turning "what if" into "here's how."

My Coding Journey

2 Years Ago

First Steps: p.js

Started my coding journey with p.js on Khan Academy - my very first program. Later that same year, discovered the magic of HTML and began creating simple webpages.

Summer 2024

First Game: Hoppy Beaver

Created my first complete game - Hoppy Beaver! Learned about game loops, collision detection, player movement, and scoring systems. This was the project that made coding "click" for me.

Summer 2025

Nexus AI Chatbot

Built Nexus AI for Khan Academy's Contest: AI! Created an interactive chatbot that could answer questions, tell jokes, and help with learning. Experimented with pattern matching and natural language concepts.

Summer 2025

Portfolio Development

Started building this portfolio/subpage while working on Nexus AI. Focused on mastering responsive design, clean UI/UX, and implementing complex animations to showcase my work.

Present

Current Focus

Expanding my skills with advanced JavaScript, exploring interactive web experiences, and pushing creative boundaries. Currently working on more complex projects while refining my design sense and problem-solving abilities.
