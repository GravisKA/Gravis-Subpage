From First Lines of Code to Interactive Projects

I'm Hampton Miles, a high school junior who discovered coding as a way to bring creative ideas to life. What started with simple HTML tags has evolved into building interactive games, AI experiments, and polished web experiences. For me, coding isn't just about writing lines—it's about solving puzzles, creating beauty in logic, and turning "what if" into "here's how."

First Hello World

Wrote my first HTML and CSS on Khan Academy. Created basic webpages with headers, paragraphs, and inline styles.

Discovering JavaScript

First Game: Hoppy Beaver

Created my first complete game using JavaScript. Learned about game loops, collision detection, and score tracking.

Built an AI chatbot for Khan Academy's competition. Experimented with pattern matching and natural language processing concepts.

Fall 2023

Portfolio Development

Started designing and coding this portfolio. Focused on clean UI/UX, responsive design, and smooth animations.

Present

Continuous Learning

Exploring advanced JavaScript, React, and backend development. Working on more complex projects and collaborating with others.
